description: 要件定義を行う
---

以下の手順で要件定義を行ってください：

1. docs/INPUT.md を確認する
   - 記載がある場合 → 内容を元に要件定義を進める
   - 記載がない/不十分な場合 → 以下を案内：
     「docs/INPUT.md に以下を記載してください：
       - 何を作りたいか
       - なぜ作りたいか
       - 誰が使うか
       - 必須機能
      記載後、再度 /project:requirements を実行してください」

2. docs/INPUT.md の内容を元に docs/PRD.md を作成

3. フォーマットは CLAUDE.md の「PRD.md テンプレート」に従う

4. README.md を更新
   - プロジェクト名（タイトル）
   - About（プロジェクト概要）
   - 背景・課題
   ※ テンプレートの説明文は削除し、プロジェクト固有の内容に置き換える

5. 作成後、内容をユーザーに確認する

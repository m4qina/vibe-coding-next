以下の手順でAPI設計を行ってください：

1. docs/DESIGN.md を確認する

2. docs/openapi.yaml を作成
   - OpenAPI 3.0 形式
   - エンドポイント定義
   - リクエスト / レスポンススキーマ

3. 作成後、内容をユーザーに確認する

4. reports/WORK_LOG.md に作業履歴を追記
   - 日付（## YYYY-MM-DD 形式）
   - フェーズ名（### API設計）
   - 実施内容（定義したエンドポイント概要）
   - 成果物へのリンク

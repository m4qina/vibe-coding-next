---

以下の手順で状況を把握し、作業を再開してください：

1. docs/ 配下のドキュメントを確認
   - PRD.md
   - DESIGN.md
   - SCREEN.md
   - openapi.yaml

2. GitHub Issue の状態を確認
   - Open / In Progress / Closed の数
   - 直近の完了タスク
   - ブロッカーがあれば報告

3. 現状をサマリーしてユーザーに報告
   - プロジェクトの概要
   - 完了している作業
   - 残っているタスク

4. 次のタスクを提案し、作業を始めるか確認する
